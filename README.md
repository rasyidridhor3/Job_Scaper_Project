

Project Python untuk mengambil data lowongan kerja dari website Fake Jobs.

## Features

- Scrape job listings
- Extract title
- Extract company
- Extract location
- Extract job URL
- Save results to CSV

## Technologies

- Python
- Requests
- BeautifulSoup4
- Pandas

## Run

```bash
python Scraper.py
```
